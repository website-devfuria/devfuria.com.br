Apesar disso não ajudar muito, vamos traduzir o acrônimo HyperText Markup Language = Linguagem de Marcação de HiperTexto.

Ajudou? Não, eu sei que não! As vezes até atrapalha, por que as pessoas pensam que HTML é uma liguagem de programação.

Não, não é! HTML é uma linguagem de marcação, ficamos marcando o texto para dizer o que é o quê.

Usamos as tags para expressar nossa marcação, temos a tag de abertura e a de fechamento.

E a palavra HiperTexto, o que significa ?

"HyperText" (HiperTexto) refere-se a links que conectam uma página a outra, seja dentro do mesmo website ou entre websites.

Você está navegando neste tutorial através de links.

Para compor um link utilizamos a tag <a> ("a" de âncora), iremos falar dela em breve.
