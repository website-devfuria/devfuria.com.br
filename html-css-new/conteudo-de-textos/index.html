Para definir um parágrafo utilizamos a tag <p>.

Para definir títulos utilizamos as tags <h1> até <h6>.

Para definir negrito utilizamos a tag <strong>.

Para quebrar linhas dentro de um parágrafo utilizamos a tag <br />
de forma isolada (sozinha) e, portanto, de auto fechamento.

Essa foi fácil! Então, vamos conhecer os contéudos de links.
